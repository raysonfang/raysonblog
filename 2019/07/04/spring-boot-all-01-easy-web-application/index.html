二.Spring Boot概述

2.1 Spring Boot是什么

Spring Boot是一个简化Spring开发的框架。用来监护spring应用开发，约定大于配置，去繁就简，just run 就能创建一个独立的，产品级的应用。

2.2 Spring Boot有哪些特点

  • 1.为所有Spring开发者更快的入门
  • 2.开箱即用，提供各种默认配置来简化项目配置
  • 3.内嵌式容器简化Web项目
  • 4.没有冗余代码生成和XML配置的要求
  • 5.提供运行时应用监控
