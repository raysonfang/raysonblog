用VBA处理Excel, 合并两个Excel文件

原创
2019-07-03
方子龙
ExcelVBA
VBA

一. 需求背景

公司需要合并两个Excel文件，这两个文件的特点就是，文件中分多个指标，需要按指标项不同，合并表格数据。

二. 方案讨论

在最开始的讨论的时候，就准备用VBA来实现，基于Excel的开发者工具， 可以自己写代码实现宏。这个缺点也很明显，就是代码维护起来比较麻烦，而且需要教他们使用，而且是面向代码，是很头痛的事。

接下来， 我自己思考的方案是用Python来做此需求，因为之前我有简单的学过， 也知道Python在处理数据方面，有很大的优势。而且语法还简单。但用Python处理Excel的实践一次也还没有，所以心里面还是很没底。准备用它来试试手。

后来， boss也安排了同事跟我一起来看看这个需求，有没有什么方案可行。准备利用现有的系统，可以做一个功能，放到系统中，而且也好维护。大家开发这个系统应该很快就能熟悉功能和代码。所以维护性还是很高，而对于用户来说，也基本上是零代码学习。

讨论了这些过后，当然得跟boss确定一下，看选用哪种，后来Boss选了VBA来实现。

问我先看看能不能做，实现起来是不是很难。我得应下来啊！

我用VBA还是两年前的事情。我自学写了个小功能， 帮公司处理每个月算大家的工时出勤时间，需要花一两天才能算清楚的，人工操作+VBA代码执行就几分钟就轻松搞定。我也问了之前的同事， 代码还在，但是我已经走了，所以就没有维护，代码就没使用了。

三. 方案实现

我下午抽空的时间，还是花时间思考了一下功能的具体实现。好在通过一番折腾，我还是用VBA实现了这个需求，而且还是很简单就可以实现操作了。

那么实现思路：打开模板文件–》选择需要合并的两个文件–》将对应地方的数据复制到模板文件中。

  • 打开文件：

  • 防止窗体抖动

  • 获取已使用的行数和列数

  • 获取单元格的值

  • 在该行下方插入一个空行，并复制数据到该行

  • 循环

  • VBA中的Function和Sub关键字的使用区别

四. 总结

虽然VBA不是我们主要的开发语言，但是我还是愿意去折腾。

技术的学习也是解决生活中的问题，解决问题的方案不止一种，使用自己最擅长的方式解决。
